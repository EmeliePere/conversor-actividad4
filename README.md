# Conversor de Monedas en C

## Descripción
Este proyecto implementa un conversor de monedas en lenguaje C.  
Permite convertir entre Quetzales (GTQ), Dólares estadounidenses (USD) y Euros (EUR) utilizando tasas de cambio aproximadas.

## Prerrequisitos
Antes de ejecutar este proyecto, asegúrate de tener instalado:
- Sistema operativo Ubuntu/Linux
- Compilador `gcc` (GNU Compiler Collection)

## Pasos para su ejecución
1. Clonar o descargar este repositorio.
2. Abrir la terminal en la carpeta del proyecto.
3. Compilar el código con:
   ```bash
   gcc conversor.c -o conversor
